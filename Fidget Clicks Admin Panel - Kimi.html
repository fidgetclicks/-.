New Features Added

1. Secret Admin Access 🔐

  • Click the logo 100 times (Fidget Clicks in the nav bar)
  • After 100 clicks, enter password: 2012!
  • You'll unlock full admin mode with a settings button in the bottom right

2. Upload Your Own Images 📸

  • In the Admin Panel: Upload product images when adding new products
  • Edit existing products: Click the upload icon on any product card
  • Edit Hero Section: Click the pencil icon on the hero image to change it

3. Change Prices & Stock 💰

  • In Admin Mode: Click the pencil icon on any product
  • Edit name, description, price, and stock directly
  • Or use the +/- buttons next to stock for quick adjustments
  • Low stock shows in red when under 10 units

4. Editable Hero Section ✏️

  • Click the pencil icon on the hero image (when admin)
  • Change:
    • Headline text (3 lines)
    • Subheadline
    • Button text (Primary & Secondary CTAs)
    • Micro label
    • Hero image

5. Product Variants/Options 🎨

  • Add multiple buying options per product:
    • Standard
    • Premium (+Keychain)
    • Bundle (3-pack)
    • Pro (+LED)
    • Limited Edition
  • Each variant can have its own price modifier
  • Customers see buttons to select their preferred option

6. Live Visitor Counter 👁️

  • Shows in bottom-left corner when in admin mode
  • Tracks every page visit
  • Displays total visitor count in real-time
  • Also shown in Admin Panel with other stats

How to Use Admin Mode

  1. Access Admin: Click "Fidget Clicks" logo in nav bar 100 times
  2. Enter Password: Type 2012! and click "Unlock Admin"
  3. Admin Panel: Click the settings gear (bottom right)
  4. Add Products: Fill form + upload image + add variants
  5. Edit Products: Click pencil on any product card
  6. Change Stock: Use +/- buttons or edit directly
  7. Edit Hero: Click pencil on hero image
  8. Logout: Click "Logout Admin" in Admin Panel

All Data Persists

  • Products, cart, hero content, and visitor count all save to browser storage
  • Refreshing won't lose your changes
  • Each visitor increments the counter automatically
